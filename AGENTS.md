## Run Instructions

### Backend (FastAPI)
1. Set your OpenAI API key in `backend/.env`:
   ```
   OPENAI_API_KEY=sk-...
   ```
2. Start the backend:
   ```bash
   cd backend && python main.py
   ```
   Runs on http://localhost:8000

### Frontend (Astro)
```bash
npm run dev
```
Runs on http://localhost:4321

Open http://localhost:4321 in your browser.
